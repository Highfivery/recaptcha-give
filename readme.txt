Quickly & easily integrate Google reCAPTCHA into your GiveWP donation forms to help prevent spam and fraudulent transactions.

== Description ==

Are you getting bombarded with fake GiveWP donations?

If your WordPress website uses the [GiveWP plugin](https://wordpress.org/plugins/give/) for donations, you're likely going to be targeted by spam bots, or fraudsters looking to test stolen credit card numbers. Stop them with Google's powerful reCAPTCHA feature.

== Installation ==

1. Upload the entire recaptcha-give folder to the */wp-content/plugins/* directory.
2. Activate the plugin through the Plugins screen (*Plugins > Installed Plugins*).
3. Visit the plugin setting to configure as needed (*Settings > reCAPTCHA for Give*).
